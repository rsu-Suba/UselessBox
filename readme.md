### 2020年

### 2月
製作開始 (2/8/2020)  
Ver1 (2/11/2020)
- Arduino Uno 3
- 蓋サーボとスイッチサーボが同じアーム  
- すべてアクリル板  
- 電池単1.5*2  

### 3月

Ver2 (3/2/2020)
- Arduino Nano Every
- 蓋サーボとスイッチサーボを分離
- すべてアクリル板
- 電池9V*1

Ver3 (3/12/2020)
- Arduino Nano Every
- Ver2の改良版
- 基盤、配線、スイッチ、マイコンを一つの基盤に移植
- すべてアクリル板
- 電池9V : 1 | 1.5V : 4

### 4～5月
アルミ土台を設計

### 7月
Ver4 (7/19/2020)
- Arduino Pro Mini
- リセットボタン、RGBLED追加
- サーボのスピード変更可にした
- アルミ土台、外側だけアクリル板
- 電池9V : 1 | 1.5V : 4
